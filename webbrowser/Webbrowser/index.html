Up – package webbrowser

Module Webbrowser

Open and reload URIs in web browsers.

Caveat

Trying to load and reload URIs from the command line in a consistant manner across browsers and operating systems seems to be a hopeless endeavour.

In particular the reload strategy mentioned below—useful to edit websites, write API documentation or web applications with Js_of_ocaml—is an indication of what should be done for what is believed to be the best user experience. However do not expect this work in all contexts (currently it only fully works with Chrome on Darwin and it is not even glitchless there).

If you know how to improve or extend the support for particular browsers and platforms get in touch here.
